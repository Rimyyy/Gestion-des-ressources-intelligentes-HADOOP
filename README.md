# Gestion-des-ressources-intelligentes-HADOOP
Hadoop, en tant que système distribué open-source, est largement 
utilisé pour stocker et traiter des volumes massifs de données. 
Cependant, bien que Hadoop soit puissant, la gestion de ses 
ressources (CPU, mémoire, et espace disque) reste un défi pour les 
administrateurs de clusters. Ce projet vise à concevoir un système 
de gestion intelligente des ressources Hadoop pour surveiller, 
optimiser et automatiser cette gestion. 
Le but principal est d'utiliser un ensemble d'outils pour créer une 
interface web capable de visualiser en temps réel les ressources 
utilisées par le cluster Hadoop. Parmi les outils utilisés, on trouve 
Putty pour les connexions distantes, SQLite3 pour le stockage de 
données, Django pour le développement de l’application web, et 
enfin, HTML et CSS pour la conception de l’interface utilisateur.
